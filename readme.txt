A small plugin that fixes ZIP file uploads in the Gutenberg Editor

== Description ==

FixZip Upload allows you to upload zip files while editing or creating a new post.

Major features in FixZip Upload include:

* Temporarily fixes this issue: https://github.com/WordPress/gutenberg/issues/23510

== Installation ==

Upload the FixZip Upload plugin to your blog, and activate it. That's it, you're done!
